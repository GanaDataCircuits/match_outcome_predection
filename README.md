# match_outcome_predection
“Machine learning project to predict match outcomes using gameplay data, achieving 85% accuracy with feature importance analysis.”

## Overview
This project predicts the winning team in a match round using machine learning.

## Problem Statement
To predict match outcomes based on gameplay data and statistics.(based on the previous rounds record we have to build a best predictive model predicting which team is gonna win a particular round)

## Approach
- Data cleaning and preprocessing
- Feature encoding
- Model training using Random Forest

## Results
- Achieved **85% accuracy**

## Key Insights
- Weapon usage strongly influences match outcomes
- Model captures gameplay patterns effectively

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
